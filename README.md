# SIMONE - Jeu de mémoire

Un jeu de mémoire classique où vous devez reproduire une séquence de couleurs qui s'allongent à chaque niveau.

## Comment jouer

1. Cliquez sur "Commencer la partie"
2. Observez attentivement la séquence de couleurs affichée
3. Répétez la séquence en cliquant sur les couleurs dans le même ordre
4. Passez au niveau suivant pour des séquences plus longues !

## Technologies utilisées

- HTML
- CSS
- JavaScript (vanilla)
